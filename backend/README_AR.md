# نظام نقاط البيع الذكي (Smart POS)

## 🔐 نظام المصادقة

- **نوع المصادقة:** Laravel Sanctum (Token-based)
- **التوكن:** يتم إنشاؤه عند تسجيل الدخول
- **الصلاحية:** يتم التحقق من الصلاحيات عبر الأدوار

---

## 📊 الأدوار الافتراضية

| الدور | الوصف | الصلاحيات |
|------|-------|----------|
| **Super Admin** | مسؤول النظام | جميع الصلاحيات |
| **Admin** | مدير إداري | معظم الصلاحيات (ما عدا الحذف الكامل) |
| **Manager** | مدير المتجر | إدارة المنتجات والمبيعات والتقارير |
| **Cashier** | كاشير | نقطة البيع والمبيعات فقط |
| **Employee** | موظف عام | عرض البيانات الأساسية فقط |

## 🚀 البدء السريع

### المتطلبات:
- PHP 8.1+
- MySQL 8.0+
- Composer
- Node.js (للواجهة الأمامية)

### التثبيت:

```bash
# 1. استنساخ المشروع
git clone https://github.com/your-repo/smart-pos-api.git
cd smart-pos-api

# 2. تثبيت الحزم
composer install

# 3. إنشاء ملف .env
cp .env.example .env

# 4. توليد مفتاح التطبيق
php artisan key:generate

# 5. تشغيل الـ Migrations
php artisan migrate

# 6. تشغيل الـ Seeders (لإنشاء البيانات التجريبية)
php artisan db:seed --class=RolePermissionSeeder

# 7. بدء الخادم
php artisan serve
```

## 🐛 حل المشاكل الشائعة

### مشكلة: "Access denied for user 'root'@'localhost'"
**الحل:**
```bash
sudo mysql -u root -e "ALTER USER 'root'@'localhost' IDENTIFIED WITH mysql_native_password BY 'root'; FLUSH PRIVILEGES;"
```

### مشكلة: "SQLSTATE[HY000]: General error: 1030"
**الحل:**
```bash
php artisan migrate:refresh
```

### مشكلة: "Class not found"
**الحل:**
```bash
composer dump-autoload
```
